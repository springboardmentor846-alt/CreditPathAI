Model Performance KPIs:

 AUC-ROC Score – how well the model separates defaulters vs non-defaulters
 Accuracy / F1-Score – classification strength
 Confusion Matrix Results – false positives vs false negatives
 Precision-Recall – especially important for imbalanced dataset

 Financial KPIs:

 Reduction in Default Rate (%)
 Recovery Rate Improvement (%)
 Portfolio At-Risk Reduction (%)
 Efficiency of recovery agents (time saved)

 Operational KPIs:

 API Response Time (FastAPI endpoint)
 Recommendation adoption rate (how many agents use suggestions)
 Number of successful predictions served
